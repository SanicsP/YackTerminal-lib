Description détaillée

Classe représentant l'entrée d'une commande de l'utilisateur lors de l'appel d'un programme.

Tout ce que l'entrée de commande peut contenir : le nom de la commande , les arguments de la commande , les spécifications suplémentaires avec leurs arguments
